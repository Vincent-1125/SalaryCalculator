# 个税计算器 — Salary Calculator

中国个人所得税薪资计算器，单文件纯前端网页，支持五险一金、专项附加扣除、年终奖单独计税、累计预扣法月度明细。

## 快速开始

直接用浏览器打开 `index.html` 即可使用。

## 功能

- **实时计算**：所有输入即改即算，无需按钮
- **年终奖单独计税**：适用年终奖独立税率表
- **累计预扣法**：1–12 月逐月个税明细表
- **五险一金**：按城市预设基数上下限，社保基数和公积金基数各自独立封顶保底（MEDIAN 夹持）
- **6 项专项附加扣除**：子女教育、继续教育、赡养老人、住房贷款利息、住房租金、婴幼儿照护
- **住房贷款/租金互斥**：两者不可同时扣除，选择贷款利息后自动忽略租金并提示
- **个人养老金**：年度上限自动限制（12,000 元/年）
- **城市预设**：北京、上海、杭州、广州、深圳，自动填充社保/公积金基数上下限
- **方案保存**：保存多套配置到 localStorage，侧边栏快速切换
- **分享链接**：所有参数编码到 URL，复制链接即可分享
- **趋势图表**：12 个月月实发与月个税 SVG 折线图
- **移动端优先**：深色主题，适配 iOS 安全区域

## 计算逻辑

基于 `SalaryCalculator.xlsx` 复刻，详见 `SalaryCalculator说明文档.md`。

### 收入

```
年工资 = 月薪 × 12 + 津贴（年）
年终奖 = 年终月基数 × 年终期数
年总包 = 年工资 + 年终奖
```

### 社保公积金

```
社保基数 = MEDIAN(月薪, 社保下限, 社保上限)
公积金基数 = MEDIAN(月薪, 公积金下限, 公积金上限)

养老(年) = 社保基数 × 8% × 12
医疗(年) = 社保基数 × 2% × 12
失业(年) = 社保基数 × 0.5% × 12
五险合计 = 养老 + 医疗 + 失业
公积金(年) = 公积金基数 × 缴存比例 × 12
```

### 个税

```
应纳税所得额（工资）= MAX(0, 年工资 - 免征额 - 专项扣除×12 - 五险 - 公积金)
应纳税所得额（年终）= MAX(0, 年终奖)

个税（工资）= 累进税率（7级超额累进，工资税率表）
个税（年终）= 累进税率（7级超额累进，年终奖税率表，速算扣除数不同）
```

> 工资和年终奖使用**不同税率表**：工资税率的速算扣除数较大（如 10% 档为 2,520），年终奖税率速算扣除数较小（如 10% 档为 210）。

### 到手

```
实际到账 = 年总包 - 五险 - 公积金 - 个税合计
含公积金到手 = 年总包 - 养老 - 失业 - 个税合计 + 公积金 + 医疗
```

> 公积金和医疗保险进入个人账户，归属个人，因此「含公积金到手」将其加回。

### 月度累计预扣

```
月扣除额 = 5000 + 月专项扣除 + (五险+公积金)/12
每月：累计应税所得 += 月薪 - 月扣除额
     累计应纳税额 = 查表(累计应税所得)
     本月纳税 = 累计应纳税额 - 上月累计应纳税额
     本月实发 = 月薪 - 本月纳税 - (五险+公积金)/12
```

## URL 参数分享

调整好参数后，直接复制浏览器地址栏的 URL 发送给对方，所有配置（月薪、扣除项、城市等）都会编码在查询参数中，对方打开即看到相同结果。
